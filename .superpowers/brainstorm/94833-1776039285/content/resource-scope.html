Which Azure resources should agentstack apply provision?

Select all that should be in scope for the first version. Click multiple options.

1

Azure Container Apps

The agent container itself — Container App Environment + Container App. Core requirement.

2

Azure Container Registry

Push built images to ACR so ACA can pull them. Alternative: use Docker Hub or another registry.

3

Azure Database for PostgreSQL (Flexible Server)

For session persistence and long-term memory. The equivalent of the Docker provider's Postgres container.

4

Azure Key Vault

Store ANTHROPIC_API_KEY and other secrets securely. Alternative: pass as ACA env vars / ACA secrets.

5

Resource Group

Create a dedicated resource group per workspace/agent. Makes cleanup easy (agentstack destroy = delete RG).

6

Virtual Network

Private networking between ACA and Postgres. More secure but adds complexity.

7

Log Analytics Workspace

Required by ACA Environment for container logs. Enables agentstack logs to work on Azure.
